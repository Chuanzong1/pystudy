# Python 学习项目集

这是一个用于记录 Python 学习过程的项目集合。每个子文件夹都是一个独立的学习项目，包含特定的学习主题和练习。

## 项目结构

```
pystudy/
├── project1/              # 项目1
│   ├── src/              # 源代码
│   ├── tests/            # 测试文件
│   └── README.md         # 项目说明
├── project2/
│   └── ...
└── README.md             # 当前文件
```

## 项目列表

每个项目都有其独立的 README 文件，详细说明了：
- 学习目标
- 涉及的知识点
- 项目描述
- 运行方式
- 学习笔记

## 如何使用

1. 每个子项目都是独立的学习单元
2. 按照项目编号顺序学习更有利于循序渐进
3. 查看每个项目的 README.md 了解详细信息
